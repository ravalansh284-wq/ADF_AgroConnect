# 🌾 AgroConnect Data Pipeline

Built in Azure Data Factory (ADF) using a Medallion Architecture, this automated pipeline ingests and transforms large-scale agricultural data to power the Harmony Agro Care ecosystem and AgroConnect API.

## 🏗️ Architecture (Medallion Pattern)
* **Bronze Layer (Raw):** Ingests raw source data (crop and soil metrics) into Azure Data Lake Storage (ADLS Gen2), preserving the original format for historical auditing.
* **Silver Layer (Cleansed):** Uses ADF Mapping Data Flows (Spark) to clean, filter, and standardize data types, outputting highly reliable datasets and bypassing complex metadata constraints.
* **Gold Layer (Aggregated):** Applies core business logic to group and summarize the data (e.g., calculating total production by district and crop). The final dataset is optimized for BI dashboards and API integration.
* **Master Orchestration:** A parent pipeline utilizes "Execute Pipeline" activities to trigger the Bronze, Silver, and Gold layers in a strict dependency chain for fully automated processing.

## 🛠️ Tech Stack
* **Orchestration & Transformation:** Azure Data Factory (Pipelines & Mapping Data Flows)
* **Storage:** Azure Data Lake Storage Gen2
* **Version Control:** GitHub

## 📂 Repository Structure
This repository contains the JSON blueprints automatically generated by Azure Data Factory:
* `/pipeline`: Orchestration logic for the Master, Bronze, Silver, and Gold layers.
* `/dataflow`: Spark-based transformation, cleansing, and aggregation logic.
* `/dataset`: Pointers to the specific ADLS Gen2 storage containers.
* `/linkedService`: Secure connection definitions for Azure resources.

## 🚀 How to Deploy
To recreate this environment:
1. Provision a new Azure Data Factory and ADLS Gen2 storage account.
2. In ADF Studio, navigate to **Manage > Git configuration**.
3. Connect to this repository and select the `main` branch. ADF will automatically import all pipelines, datasets, and data flows onto your canvas.
